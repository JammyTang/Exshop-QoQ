# Exshop-QoQ

1. 請在雲端硬碟建立名為『exshop』的資料夾，
      接著在其中建立名為『QoQ』的資料夾，
      最後在其中建立名為『raw_data』與『data』的兩個資料夾

2. 點選『Exshop QoQ_analytics』右上方，選擇〈更多動作〉->〈建立副本〉，
      並將建立好的檔案副本放在 QoQ 資料夾中

3. 請將從神蹟下載好檔案重新命名，以下為命名規範：
    - 本季：new / 上一季：old
    - 官網：shopline / 蝦皮：shoppe / 門市：store / B2B：b2b

舉例：本季門市檔案名稱應為：new_store

4. 請將命名好的 所有 Excel 檔放在〈raw_data〉 資料夾中
